This folder contains all of the raw data that was used in this project as well as the data dictionaries. Two datasets were used for the School System Financial Data 2017 -- School_Data_Census
and School_Data_Census_Extra. 

Sources: 
Finance and Insurance Summary Statistics: https://www.census.gov/data/tables/2017/econ/economic-census/naics-sector-52.html
School System Financial Data 2017: https://www.census.gov/data/tables/2017/econ/school-finances/secondary-education-finance.html
School Data for Classification: https://www.kaggle.com/noriuk/us-education-datasets-unification-project?select=states_all_extended.csv
